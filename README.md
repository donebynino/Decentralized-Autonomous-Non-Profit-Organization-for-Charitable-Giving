# Charitable DAO

A decentralized autonomous organization (DAO) for transparent charitable giving built on the Stacks blockchain. This platform enables donors to contribute sBTC to a shared treasury, receive governance tokens, and collectively decide how funds are allocated to charitable projects.

## 🎯 Mission

Transform charitable giving through blockchain transparency and donor governance. Every donation, vote, and fund distribution is recorded on-chain, ensuring complete accountability and giving donors direct control over how their contributions are used.

## ✨ Key Features

### For Donors
- **Transparent Donations**: Contribute sBTC and receive governance tokens proportional to your donation
- **Direct Governance**: Vote on funding proposals from charitable projects
- **Complete Visibility**: Track exactly how your donations are being used
- **Delegation Options**: Delegate voting power to trusted community members

### For Charitable Projects
- **Proposal Submission**: Request funding with detailed project descriptions and timelines
- **Automatic Funding**: Approved proposals receive funds automatically from the treasury
- **Community Validation**: Projects are vetted by the donor community

### For Everyone
- **Full Transparency**: All transactions, votes, and decisions are publicly viewable
- **Audit Trail**: Complete history of donations, proposals, and fund distributions
- **Decentralized**: No central authority controls the funds or decisions

## 🔄 How It Works

1. **Donate**: Contributors send sBTC to the DAO treasury
2. **Govern**: Donors receive governance tokens based on their contribution amount
3. **Propose**: Charitable projects submit funding proposals with detailed plans
4. **Vote**: Token holders vote on which projects should receive funding
5. **Fund**: Approved projects automatically receive sBTC from the treasury
6. **Track**: All activities are recorded on-chain for complete transparency

## 🛠 Technology Stack

- **Blockchain**: Stacks blockchain for smart contracts and transparency
- **Smart Contracts**: Clarity language for secure, predictable contract execution
- **Currency**: sBTC for donations and funding
- **Development**: Clarinet for local development and testing
- **Testing**: Vitest with TypeScript for comprehensive test coverage

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Clarinet](https://github.com/hirosystems/clarinet) for Stacks development
- [Git](https://git-scm.com/) for version control

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd charitable-dao
```

2. Install dependencies:
```bash
npm install
```

3. Start local development environment:
```bash
clarinet console
```

### Running Tests

```bash
# Run all tests
npm test

# Run tests with coverage report
npm run test:report

# Watch for changes and re-run tests
npm run test:watch
```
